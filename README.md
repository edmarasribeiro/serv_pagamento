Para rodar o projeto, abra o terminal e clone o repositório com o comando git clone https://github.com/edmarasribeiro/serv_pagamento.git. 
Em seguida, acesse a pasta do projeto digitando cd serv_pagamento. 
Instale o framework de testes Mocha rodando o comando npm install e, 
por fim, execute toda a suíte de testes automatizados com o comando npm test.
